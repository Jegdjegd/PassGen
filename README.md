# 🔐 PassGen — Secure Password Generator

> Aplicación web moderna para generación de contraseñas seguras, con foco en UX, rendimiento y escalabilidad.

🌐 **Live Demo:** https://passgenjg.vercel.app/  
📦 **Repository:** https://github.com/Jegdjegd/PassGen  

---

## 🧩 Project Overview

**PassGen** es una aplicación frontend desarrollada para generar contraseñas seguras de forma rápida y configurable.  

El objetivo principal es ofrecer una herramienta ligera, intuitiva y eficiente que permita a cualquier usuario mejorar su seguridad digital sin fricción.

Este proyecto refleja buenas prácticas en desarrollo frontend moderno, incluyendo arquitectura limpia, diseño responsive y experiencia de usuario optimizada.

---

## 🚀 Key Features

- Generación de contraseñas con múltiples configuraciones:
  - Alfanuméricas
  - Numéricas + caracteres especiales
  - Combinación completa (alta entropía)

- Longitud configurable:
  - Rango de 8 a 64 caracteres

- Regeneración instantánea

- Copia al portapapeles en un solo clic

- Interfaz responsive (adaptada a escritorio y móvil)

---

## 🛠️ Tech Stack

- **React** — UI basada en componentes  
- **Vite** — entorno de desarrollo rápido y optimizado  
- **TailwindCSS** — diseño moderno y eficiente  

---

## 🧠 Technical Highlights

- Generación de contraseñas completamente en cliente (client-side), sin dependencias externas  
- Lógica modular para facilitar escalabilidad futura  
- UI optimizada para interacción rápida y mínima fricción  
- Diseño responsive implementado desde el inicio (mobile-first mindset)  

---

## 🔮 Roadmap

Este proyecto está planteado con una evolución clara hacia un producto más completo:

- Sistema de autenticación de usuarios
- Bóveda de contraseñas (secure storage)
- Historial de generación
- Aplicación nativa para Android
- Exploración futura de generación basada en principios de computación cuántica

---

- Desarrollo frontend moderno con herramientas actuales
- Capacidad de construir interfaces limpias y funcionales
